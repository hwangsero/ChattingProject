# kaka

## 프로젝트 요약
### 개발기간
2018/12/10 ~ 2019/01/17

### 참여인원
3명

### 개요
소켓 통신을 이용한 채팅 사이트 

### 주요기능
- 아이디를 이용해 친구 검색 및 친구 추가
- 등록된 친구들 중에서 대화할 상대를 선택하여 대화방 개설
- 개설된 대화방에서 여러명의 친구들과의 실시간 채팅 기능
- 채팅방목록에서 채팅방들의 가장 최근 대화내용 확인 및 최근에 채팅이 이루어진 순서대로 채팅방 목록 실시간 변화
